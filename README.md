### 📋 Sobre o Projeto
O **MK Design** é uma Landing Page de alto padrão desenvolvida para um escritório de design de interiores especializado em móveis planejados e projetos luminotécnicos. O layout foi construído com uma estética *dark mode* e detalhes em dourado para transmitir exclusividade e sofisticação.

Este projeto foca na conversão direta via WhatsApp e na apresentação visual de serviços de luxo.

---

### 🛠️ Tecnologias e Recursos
Para este projeto, foquei em uma implementação rápida, porém extremamente elegante:

- **HTML5 Profissional:** Estrutura focada em acessibilidade e seções bem definidas.
- **Tailwind CSS:** Utilizado para estilização responsiva e moderna.
- **Google Fonts:** Integração das fontes *Poppins* e *Roboto* para legibilidade premium.
- **JavaScript Vanilla:** Lógica para menu mobile, scroll suave e animações de *fade-in* ao rolar a página.
- **UX/UI:** Uso de contrastes altos e elementos visuais que remetem ao mercado de luxo.

---

### ✨ Funcionalidades Principais
* **Animações On-Scroll:** Elementos que surgem suavemente conforme o usuário navega.
- **Menu Mobile Inteligente:** Navegação adaptada para dispositivos móveis.
- **Integração com WhatsApp:** Botões de call-to-action (CTA) estrategicamente posicionados.
- **Design Responsivo:** Visual impecável desde smartphones até monitores ultra-wide.

---

### 🚀 Como visualizar o projeto

1. Faça o download dos arquivos.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Não é necessário instalar dependências, pois o estilo é carregado via CDN.
